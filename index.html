圖片批次轉 PDF

可選 A1 / A2 / A3，直式或橫式 · JPG/PNG 無損嵌入原圖（HEIC 轉高品質 JPEG）· 全程在瀏覽器本機處理，圖片不上傳
點此選擇圖片，或把圖片拖進來
支援 JPG / PNG / HEIC / WebP / AVIF / GIF / BMP，可一次選多張
關於畫質與格式
  • JPG / PNG：直接把原始檔案位元組嵌入 PDF，完全無損，與原檔 100% 相同。
  • WebP / AVIF / GIF / BMP：由瀏覽器解碼後以 PNG 無損重新封裝再嵌入，像素資料不變。
  • HEIC：PDF 容器本身不支援 HEIC、瀏覽器也無法直接保留它，所以 HEIC 必須先解碼再轉成高品質 JPEG（用最高品質 0.95，視覺上近乎無損）才能嵌入。
